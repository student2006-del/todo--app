# TaskTracker

## Проблема
Багато студентів та користувачів забувають про свої завдання та дедлайни, що призводить до стресу та втрати продуктивності.

## Пітч
**TaskTracker** дозволяє швидко фіксувати завдання, відстежувати їх виконання та бачити всю історію у зручному списку.  
Цільова аудиторія – студенти та активні користувачі, яким потрібно організовувати свій час.  
Результат – всі завдання доступні в одному місці, їх можна позначати як виконані без зайвих кроків.

## Ядро ідеї / MVP
- Додати нове завдання
- Перегляд завдань у списку
- Позначення завдання як виконаного

## Стейкхолдери
- **Користувачі:** сприймають цінність через зручність організації завдань  
- **Оцінювачі:** перевіряють, чи продукт допомагає ефективніше виконувати завдання  
- **Впливові особи:** розробники або адміністратори, які можуть обмежувати ресурси

## Мета першої ітерації
Забезпечити базову функціональність для додавання, перегляду та завершення завдання

## Ознаки успіху
- Головний сценарій (додавання → перегляд → виконання) можна виконати без зайвих кроків
- Операції стабільно працюють протягом тижня використання
- Завдання успішно позначаються як виконані та відображаються у списку
